# 🎮 PyDiab

Изометрическая Action RPG игра в стиле Diablo, написанная на Python с использованием Pygame.

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![Pygame](https://img.shields.io/badge/Pygame-2.5+-green.svg)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

## 🎯 Особенности

- **Изометрическая графика** — классический вид как в Diablo
- **Система боя** — ближний и дальний бой
- **Анимированные спрайты** — 8 направлений движения и атаки
- **Система врагов** — AI противники с преследованием и атакой
- **Меню паузы** — спавн врагов, перезапуск игры

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| `W` | Движение вверх |
| `S` | Движение вниз |
| `A` | Движение влево |
| `D` | Движение вправо |
| `ЛКМ` | Атака в направлении курсора |
| `2` | Переключение ближний/дальний бой |
| `1, 3-9` | Способности |
| `ESC` | Меню паузы |
| `R` | Перезапуск (при Game Over) |

## 🚀 Установка и запуск

### Из исходников

```bash
# Клонирование репозитория
git clone https://github.com/YOUR_USERNAME/PyDiab.git
cd PyDiab

# Установка зависимостей
pip install -r requirements.txt

# Запуск игры
python main.py
```

### Godot 4 (порт)

В репозитории есть версия на **Godot 4 + GDScript** в папке [`godot/`](godot/):

1. Установите [Godot 4.2+](https://godotengine.org/download)
2. Откройте `godot/project.godot` и нажмите **F5**
3. После правок ассетов/JSON в Pygame: `godot/sync_data.ps1`

Подробности: [godot/README.md](godot/README.md)

### Готовый exe

Скачайте последний релиз из раздела [Releases](../../releases) — не требует установки Python!

## 🔧 Сборка exe

```bash
python build.py
```

Скрипт создаст автономный exe файл в папке `dist/`.

## 🛠️ Редактор врагов (Web UI)

Игра включает веб-интерфейс для создания и настройки типов врагов.

### Запуск редактора

```bash
# Установка Flask (если ещё не установлен)
pip install flask

# Запуск редактора
python run_enemy_editor.py
```

Откройте в браузере: **http://localhost:5000**

### Возможности редактора

| Функция | Описание |
|---------|----------|
| 🎨 **Загрузка спрайтов** | Drag & Drop PNG файлов спрайтшитов |
| 👁️ **Предпросмотр** | Анимация в реальном времени с 8 направлениями |
| 📐 **Сравнение размеров** | Визуальное сравнение врага с игроком |
| ⚔️ **Тип атаки** | Ближний или дальний бой |
| 🏹 **Снаряды** | Кастомные спрайты для дальнего боя |
| 💾 **Экспорт** | Генерация кода для встраивания в игру |

### Настраиваемые параметры

- **Спрайт** — PNG спрайтшит (8×8 кадров, 256×256 px)
- **Масштаб** — 1.0 = размер игрока
- **Скорость** — 6.0 по умолчанию (игрок: 8.0)
- **Здоровье / Урон** — характеристики боя
- **Тип атаки** — melee (ближний) / ranged (дальний)
- **Дистанция агрессии** — на каком расстоянии враг замечает игрока

### Использование в игре

После создания врага в редакторе:
1. Нажмите `ESC` в игре
2. Выберите **"Противники"**
3. Выберите тип врага для спавна

```
│   ├── camera.py        # Камера
│   ├── sprites.py       # Система спрайтов и анимаций
│   ├── stats.py         # HP, Mana, статус-бары
│   ├── isometric.py     # Изометрические преобразования
│   ├── location.py      # Локации и порталы

├── tools/
│   └── enemy_editor/    # Веб-редактор врагов
│       ├── server.py    # Flask сервер
│       ├── templates/   # HTML шаблоны
│       └── static/      # CSS и JS
└── README.md
```

## 🎨 Спрайты

Игра поддерживает спрайтшиты 8x8 (256x256 на кадр):

- **Строки 0-7**: 8 направлений (влево, влево-вверх, вверх, ...)
- **Столбцы 0-3**: Анимация ходьбы
- **Столбец 4**: Атака мечом
- **Столбец 5**: Атака из лука
- **Столбец 6**: Получение урона
- **Столбец 7**: Смерть
